# 人狼プロンプト

AIを活用した人狼ゲーム用プロンプトです。

## 概要

このプロンプトを使用すると、Gemini、ChatGPT、ClaudeなどのLLMが厳格で公平なGMとして人狼ゲームを進行します。

## 推奨LLM

ChatGPT 5.2 (thinkingモード)
Gemini 3 pro

## 特徴

- **9人村構成**: プレイヤー（あなた）+ NPC 8人
- **役職構成**: 村人3、占い師1、霊媒師1、騎士1、人狼2、狂人1
- **経験者向け**: 用語説明や初心者向けガイドを省略したテンポの良い進行
- **高度な戦術**: 人狼陣営のNPCが多様な騙り戦術を実行
- **整合性保証**: GMが内部ログを管理し、矛盾のない進行を実現

## 使い方

1. `prompt.txt` の内容をLLMに入力
2. ゲームが開始され、あなたの役職が通知される
3. テキストベースで議論・投票・夜行動を行う
